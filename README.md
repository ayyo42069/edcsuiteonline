# EDC15P+ Web Editor

A fun side project to load, visualize, and modify **EDC15P+** ECU files directly in the browser. It serves as a free, open-source alternative to paid online suites.

### ⚠️ Mappacks Needed
The project currently has a limited set of mappacks compared to paid online versions.
* **Contributions are welcome:** If you have definitions, please submit a PR to help expand the database.
